粵拼輸入法（iOS）私隱佈告


我哋唔會收集或者分享用戶個人資訊
呢句話概括咗我哋嘅私隱政策。
下邊係更多嘅細節詳述。

主應用程式

主應用程式唔會記錄或收集任何個人資訊。
主應用程式唔會傳送任何資訊返我哋嘅伺服器（服務器），因爲我哋根本冇伺服器。
主應用程式使用系統提供嘅標準控件 SFSafariViewController 來開啓網鏈。

鍵盤

鍵盤嘅絕大部分功能都毋需開啓「全權取用」。
無論開啓「全權取用」與否，鍵盤都全程唔會訪問網絡。
無論開啓「全權取用」與否，鍵盤同主應用程式之間都唔會進行任何交流 / 分享。

爲咗實現用戶本地詞庫，鍵盤會喺設備本地記錄用戶選擇嘅候選詞。
記錄條目包括：候選詞漢字、候選詞粵拼、候選詞使用頻率。
鍵盤只會喺粵拼模式下做記錄，喺輸入數字、符號、英文嘅時候唔會記錄任何嘢。

依照 iOS 沙盒機制，鍵盤保存喺本地嘅用戶詞庫，唯有鍵盤自己可以接觸到。
若果用戶喺設備度開啓咗 iCloud 備份，鍵盤保存嘅用戶詞庫會備份到 iCloud。
用戶可以自行到系統設定 App 嘅 iCloud 備份頁面，關閉「粵拼輸入法」備份。

全權取用

由於 iOS 系統限制， 鍵盤有兩個功能需要用戶開啓「全權取用」權限：
• 鍵盤上部工具欄嘅「貼上」掣。
• 撳掣 Haptic 振動反饋。
若果用戶冇開啓「全權取用」，呢兩個功能會無法使用；除此之外，唔會有其它影響。

開源

本輸入法一直同步開放源碼於 GitHub：yuetyam/jyutping 。

Privacy Notice of Jyutping for iOS


We do NOT collect or share personal information.
That's our privacy policy in a nutshell.
The rest of this page is going to explain more the detail.

The main App

The main App can not collect any personal information.
The main App can not send any data back to our servers since we don't even have a server.
The main App uses the standard SFSafariViewController to open web links.

The Keyboard Extension

The Keyboard Extension does not require Full Access.
The Keyboard Extension does not collect any personal information.
There is no communication between the Keyboard Extension and the main App.

Open Source

This App is fully open source on GitHub: yuetyam/jyutping.
