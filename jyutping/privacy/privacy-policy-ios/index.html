Privacy Notice of Jyutping for iOS


We do NOT collect or share personal information.
That's our privacy policy in a nutshell.
The rest of this page is going to explain more the detail.

The main App

The main App can not collect any personal information.
The main App can not send any data back to our servers since we don't even have a server.
The main App uses the standard SFSafariViewController to open web links.

The Keyboard Extension

The Keyboard Extension does not require Full Access.
The Keyboard Extension does not collect any personal information.
There is no communication between the Keyboard Extension and the main App.

Open Source

This App is fully open source on GitHub: yuetyam/yutping.
